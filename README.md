# Скрипт для исправления электронного дневника

Этот скрипт позволяет автоматически исправлять плохие оценки, удалять замечания и добавлять 
похвалы в системе электронного дневника.

***

## Как установить

Прежде чем начать работу со скриптом необходимо установить библиотеки из файла находящемся 
в репозитории электронного дневника `requirements.txt`.

Для установки необходимых библеотек использующихся в скрипте, в консоли 
напишите команду и дождитесь установки всех необходимых для работы репозитория 
библиотек:

```
$ pip install -r requirements.txt
```

Документацию к библиотекам можете изучить на сайте, найдя нужную [PyPI](https://pypi.org/).

***

## Как использовать

1. До начала работы со скриптом, найдите и скачайте репозиторий с сайтом электронного дневника. 
После скопируйте в корневую папку данный скрипт.

2. Настройте переменные в репозитории под себя и поднимите сервер командой:

```
$ python manage.py runserver
```

3. Откройте терминал в папке с проектом и выполните:

```
$ python manage.py shell
```

4. Использование функций:

Чтобы использовать функции импортируйте скрипт в консоли командой

```
$ import script
```

* `fix_marks('ФИО')` Исправляет плохие оценки ученика.
* `remove_chastisements('ФИО')`  Удаляет все замечания ученика
* `create_commendation('ФИО', 'Название предмета')` Добавиляет похвалу от учителя

5. Примеры использования:

```Python
fix_marks('Фролов Иван')  # Исправить свои оценки
remove_chastisements('Фролов Иван')  # Удалить свои замечания
create_commendation('Фролов Иван', 'Музыка')  # Добавить похвалу
```

В конце не забудьте обновить страницу сайта с электронным дневником и убедиться, что всё исправлено. 
Если возникнет ошибка скрипт сам подскажет что делать, читайте сообщения в консоли!
